PyRASA | Spectral parametrization in python based on IRASA

  • a package for spectral parametrization in python based on the IRASA algorithm
  • separates time series data in their periodic and aperiodic components and facilitates their analysis
  • [Code]
  • [Documentation]


AlmKanal | Automated preprocessing pipelines for M/EEG data

  • a package that allows its users to build automatic M/EEG preprocessing pipelines
  • AlmKanal is built on top of MNE Python and can be easily extended
  • contains a number of template pipelines e.g. for resting state M/EEG data
